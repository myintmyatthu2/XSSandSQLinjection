SQL Injection Tutorial

Warning: This application contains intentional SQL injection vulnerabilities for educational purposes.

What is SQL Injection?

SQL Injection is a code injection technique that might destroy your database. It is one of the most common web hacking techniques.

How SQL Injection Works

SQL injection occurs when an attacker can insert malicious SQL code into a query by manipulating user input.

Try It Out

Visit the Admin Review page and try these payloads in the search box:

  • ' OR 1=1 -- - Show all records
  • ' UNION SELECT username, password, NULL FROM users -- - Extract user credentials
  • '; DROP TABLE feedback; -- - Attempt to delete table (protected)

Prevention

  • Use parameterized queries (prepared statements)
  • Validate and sanitize all user input
  • Use principle of least privilege for database accounts
  • Implement proper error handling (don't expose database errors)

You can toggle SQL injection protection in the Security Dashboard to see the difference.
